容器是微服务的最佳载体，Kubernetes 是微服务的最佳运行平台，[Istio](https://istio.io) 是 Kubernetes 上最佳的 service mesh。

2017年4月 Docker 项目改名为 moby，10月的 DockerCon 在哥本哈根 Docker 公司宣布拥抱 Kubernetes，对于容器编排领域 Kubernetes 已经成为事实上的王者。
